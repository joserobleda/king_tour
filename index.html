Dialog positioning

The dialog can easily be placed around an element. We go around clock-wise

To place a dialog each tour step-div has an option "pos" in its title tag

"pos" is a tripple defining the position like this:

ltb - left-top-bottom

1. Dialog position:

t(op), r(ight), b(ottom), l(eft)

2. Arrow position:

t(top), m(middle), b(ottom)

OR with dialog on top/bottom

l(left), c(enter), r(right)

3. Dialog alignment:

t(op), m(middle), b(ottom)

OR with dialog on top/bottom

l(left), c(enter), r(right)

tll - top-left-left

tlc - top-left-center

tlr - top-left-right

tcl - top-center-left

tcc - top-center-center

tcr - top-center-right
